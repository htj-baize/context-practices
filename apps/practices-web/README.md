# Practices Web

Shared web validation shell for `context-practices`.

## Intent

- keep case logic and artifact generation in `cases/*`
- keep shared presentation, interaction, and mobile validation UI in one app
- avoid rebuilding one-off static demos per case

## Stack

- Next.js
- React
- Tailwind CSS v4
- shadcn-style component structure

## Local Development

From repository root:

```bash
pnpm install
pnpm dev
```

Open [http://127.0.0.1:3000](http://127.0.0.1:3000).
